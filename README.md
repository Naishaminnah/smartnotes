# Smart Notes

A full-stack Django web application for managing personal notes with authentication, search, and modern UI.

## Features

- User Authentication (Login / Signup / Logout)
- Create, Edit, Delete Notes (CRUD)
- User-specific notes (secure)
- Search functionality 
- Dark mode
- Responsive UI using Bootstrap

## Tech Stack

- Backend: Django
- Frontend: HTML, Bootstrap
- Database: SQLite

## Installation

```bash
git clone https://github.com/Naishaminnah/smartnotes.git
cd smartnotes
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
